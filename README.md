# Real Estate CRM Landing Page

A responsive Next.js landing page recreated from the supplied Builder.io/Figma design for a real estate CRM product.

## Getting started

```bash
npm install
npm run dev
```

Open `http://localhost:3000` in your browser.

## Main files

- `app/page.tsx` — page structure and content
- `app/globals.css` — Next.js global style entrypoint
- `styles.css` — responsive landing-page styles
- `app/layout.tsx` — metadata and Inter font configuration

The page includes the hero dashboard, feature grid, workflow timeline, screenshots, automation map, target audiences, sales CTA, contact form, and footer.
